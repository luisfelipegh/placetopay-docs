## Cancelar un AutoPago {{ id: 'cancel-autopay', tag: 'POST', label: '/api/client/cancel'}}
Su función es cancelar/deshabilitar un autopago para que no se realicen los cobros de forma automatica/recurrente programados con anterioridad.

<Note type="warning"> Una vez cancelado el autopago, este no podrá ser reactivado, en caso de necesitar nuevamente un autopago se deberá crear uno nuevo. </Note>
